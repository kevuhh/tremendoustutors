change folders
cd .. // go back one level up
cd / // go to root folder(highest folder)
cd [name] //go into folder

ls // list all contents in folder
rm [name] // remove file
mv [oldname] [newname] // move old file to new file or from old folder to new folder
cp [oldname] [newname] // copy file to new name or location

Start Server
sudo systemctl start tutors
Stop Server
sudo systemctl stop tutors
Recycle Server
sudo systemctl restart tutors
